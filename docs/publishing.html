Chapter 9 Publishing

9.1 Sharing your Rmd files via Github pages

  1. Save your analysis in a file index.Rmd

  2. Click ‘knit’ in RStudio
  • (selecting ‘knit to HTML’ if you haven’t specified this in the header)

  1. Save your files to a github repo: https://github.com/username/myrepo/

  2. Via the browser, edit the settings for the repo and scroll down to ‘Github Pages’. There change ‘Source’ to ‘Master’

Your pages will be at https://username.github.io/myrepo/
